## Contribution Guide

### Prerequisites
```sh
- Fork the repo
- Clone the fork repo to your local
- Create a new branch for your coming PR. e.g. fix-bug-xxxx
```

### Start
```shell script
# install dependencies
- yarn run bootstrap
# build project & link the cli to global by npm link
- yarn run pre:start
# start local serve
- yarn run start
```

### Pull Request

```shell script
- Send the branch to your remote
- Create pull request 
```
